# TACO KING 🌮
タコスをまだ食べたことがない人や若い世代に「楽しそう・おいしそう」と思ってもらうことをコンセプトに制作したランディングページです。
カラフルな色遣いとアニメーションで、見ているだけで気分が上がるPOPな雰囲気を意識しました。

## Features
- タコスを中心に食材が円周上を回転するアニメーション
- スクロールに応じた要素のフェードイン演出
- メニュー画像の無限ループスライダー
- メニューページ・お問い合わせページの実装(複数ページ構成)
- ハンバーガーメニューによるスマホ対応

## Challenges
- 食材を円形に等間隔配置するため、JavaScriptでCSSの`transform`(rotate と translate の組み合わせ)を動的に設定する処理を実装
- 画面サイズが変わっても食材との距離が不自然にならないよう、半径をタコス画像の幅から計算し、配置位置を動的に決定
- メニュー画像をシームレスにループさせるため、JavaScriptで最初の数枚を複製して末尾に追加し、CSSアニメーションと組み合わせて自然な無限スクロールを実現
- 見出しを1文字ずつアニメーションさせる演出として、文字を`<span>`に分解し、スクロールで画面に入ったタイミングで順番にフェードインする処理を実装
- スクロール位置に応じた要素の表示制御として、`getBoundingClientRect`で各セクションの位置を監視し、画面内に入ったタイミングで表示するよう実装

## What I Learned
- CSSの`transform`を組み合わせて要素を円形に配置する手法
- JavaScriptで動的な計算を行い、画面サイズに依存しないレイアウトを作る重要性
- アニメーション処理をJavaScriptとCSSで役割分担する考え方(JSで配置・トリガー、CSSで動き)
